# 🎵 Spotify Analytics Dashboard | Power BI

An interactive Spotify Analytics Dashboard built using **Power BI**, **Power Query**, and **DAX** to analyze the Spotify Top 50 World Songs dataset. The dashboard provides interactive insights into songs, artists, popularity, albums, and release trends through a clean Spotify-inspired user interface.
